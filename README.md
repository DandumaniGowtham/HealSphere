## 🧠 About HealSphere

HealSphere is a **full-stack healthcare management system** designed to manage hospitals, doctors, and patients with real-time analytics and secure authentication.

It provides:

* 🏥 Hospital Management
* 👨‍⚕️ Doctor Management
* 🧑 Patient Tracking
* 📊 Dashboard Analytics
* ⭐ Reviews & Ratings
* 🔐 Secure Authentication (JWT)

---

## ✨ Core Features

### 🖥️ Frontend (React)

* ⚡ Modern UI with Vite
* 🔍 Search hospitals & doctors
* 📊 Dashboard with charts
* 📈 Admissions & discharge analytics
* ⭐ Reviews system
* 🔐 Authentication UI

---

### ⚙️ Backend (Spring Boot)

* 🔗 RESTful APIs
* 🔐 JWT Security
* 🧠 Layered Architecture (Controller → Service → Repository)
* 🗄️ MySQL Database
* 📦 Modular code structure

## ⚙️ Setup Instructions

### 🔹 Clone

```
git clone https://github.com/DandumaniGowtham/HealSphere.git
cd HealSphere
```

---

### 🔹 Frontend

```
cd HealSphere-Frontend
npm install
npm run dev
```

---

### 🔹 Backend

```
cd HealSphere-Backend
mvn clean install
mvn spring-boot:run
```

---
